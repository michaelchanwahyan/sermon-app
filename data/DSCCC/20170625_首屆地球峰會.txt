一、引言 世界環境日

聯合國大會1972年通過成立聯合國環境署UNEP並在斯德哥爾摩 (Stockholm) 召開人類環境會議Conference on Human Environment，決定6月5日為「世界環境日」。其目的是鼓勵全球民眾環境保護意識並採取環保行動。

由1974年開始世界各地都舉辦活動。

香港的教會最近幾年才有限地響應有關活動和講道，但不少教會還不知道有環境日，也少關心和人與環境的關係，這不只是與物質世界的關係，而是與上帝的關係。

二、地球峰會

1972年後20年來聯合國發現地球生物、生態、生境遭到極大的人為破壞，情況持續惡化；於1992年再召開更大型的會議。由聯合國環境與發展會議 (UNDEP), (是UNEP，UNDP聯合的大會 ) ，在6月3-14在巴西里約熱內盧Rio Summit召開世界地球峰會。172個國家出席，其中有116位國家元首。如果全球有196的國家的話，共有88%出席。共有2,400非政府組織參加(NGO) 。

會議的原因，是囚人類的命運受到環境破壞而面臨險境，包括:
1. 地球暖化─大量使用化石燃料，交通氣體排放，空氣污染
2. 生物絕種，因生境遭破壞
3. 森林的過度砍伐和
4. 過度開發，都市化和污染日增。

會議的成果：建立兩個國際公約 Convention ( 約Covenant)

生物多樣性公約 (Convention of Biological Diversity CBD)
聯合國氣候變化框架公約 (UN Framework Convention on climate Change UNFCCC)
這兩個協約都影響中國和香港
後者發展成為京都議定書Kyoto Protocol (1997)。192國家簽署，2005年生效，限制(二氧化碳) Green House Gas GHG的排放。引至 (2015) 巴黎氣候協議
這次峯會的特點是：
1. 由世界組織召開
2. 由各國領袖出席、 有關專家、有權人士
3. 是面對危機的緊急時刻
4. 建立「公約」大家遵守
5. 有標誌，記號以防忘記

其實世界上第一屆的地球峰會不是1992年的巴西Rio Summit而是遠古之年，由上帝親自召開的地球峰會。

C. 「約」的內容和性質 ( 9:11 )
這段經文內出現七次 「約」字，顕示重要！
(a) 單方面的由上帝承諾，保證；
(b) 不再以洪水毀壞地；
(c) 人不須承擔任何責任和工作；
(d) 是普世的，關乎全人類和地球；
(e) 是永久性的( 9:12；16 ) 沒有時限的；
(f) 是有記號的( 9:13-14；16 ) ；
(g) 上帝會記念的，放在心中，不會吃言。

這是聖經中第一個上帝與人所立的約；是造物者與被造者之間的約；
是至高無上的，是首個也是關乎普世的。
在舊約還有三個約，分別是上帝與亞伯拉罕、摩西，大衛所訂的約
但性質不同：
(a) 亞伯拉罕( 創17:1-14 )成為選民，以割禮為記號
(b) 摩西之約 ( 出31:16-17 ) 以安息日為記號
(c) 與大衛之約( 撒下:1-17 ) 應允國位堅立
(d) 新約，耶穌設立聖餐( 路12:20 ) ，酒和餅，這杯是用我血所立的新約。

約是承諾，有約束力，有履行的責任和義務。
  上帝採取主動促成這次首屆地球峰會，意義重大。

上帝承諾

(a)凡有血肉的不再被洪水滅絕；

(b.)也不再有洪水毀壞地( 9:11 ) 。

問題：  上帝不用洪水，會不會用其他方法？
以後所多瑪和蛾摩拉，被天火所焚是局部的，但其理由相同，因為人犯罪。但是上帝不毀滅，人會不會自行毀滅？情況極為可能。

                     (b) 天然，美麗；

                      (d) 沒有地域之限。

四、首屆地球峯會召開的背景

 

首屆峰會是在洪水退去後召開，人和生物出了方舟，挪亞築壇獻祭，

結束了一年零10天的洪水淹地 (由2月17日(7:11)至次年二月27日(8:14))，挪亞由600歲到601歲。

耶和華說出心聲：“我不再因人的緣故咒詛地，(人從小時心裏懷著惡念)，也不再按著我才行的，滅各種活物了。”(8:21)

大地恢復自然的秩序：地還存留的時候、稼穡、寒暑、冬夏、晝夜、就永不停息了(8:22) 。

為甚麼耶和華要用洪水毀滅大地和生物呢？

    因為人的罪

   6:5  耶和華看見人在地上罪惡很大，終日所思想的盡都是惡

   6:11 世界在神面前敗壞，地上滿了強暴 corrupted full of violence

   6:12 神觀看世界，見是敗壞了；凡有血氣的人在地上都敗壞了行為

   6:13 因為地上滿了他們的強暴。

  

上帝創造了人地和其中的生物，祂關心，愛護，並非創造完成後不理，而是時常查看，視察。見到情況令祂失望：

(1) 地上罪惡很大；

 (2) 世界在神面前敗壞；與神的創造原意相反；

    (神是有期望的，有公義的，容不下罪惡 ) ；

(3)人的思想敗壞；

(4)終日所思盡都是惡 - 不停止的，習以為常的，不是偶然的；

(5)有血氣的人在地上敗壞了行為；不但思想還有行動；

(6)地上滿了強暴( 6:11；13 ) ；侵害和欺壓別人和生物。

如該隱殺亞伯 － 無法無天，堆泥，斬樹，破壞生態，強橫無理，喜歡就去做。如所多瑪，蛾摩拉城邑的人，他們一味的行淫，隨從逆性情慾。滿眼是淫色，止不住犯罪。( 彼後2:14 )

上帝的反應：

(a)後悔造人在地上，心中憂傷 ( 6:6 ) ；因為我造他們後悔了 ( 6:7 )

後悔，失望，不及期望，遠離原意，有負所托。

(b)說要將所造的人和走獸，並昆蟲，以及空中的飛鳥，都從地上除滅。

(c)因為地上的強暴，凡有血氣的人；他們的盡頭已經來到我面前

(d)我要把他們和地一併毀滅 ( 6:13 ) 。

世界敗壞，充滿暴力，惡人橫行，有違聖意；令造物者上帝失望，後悔，本以為可以遍滿地面，由人代為管理，雖知敗壞無遺。達不到創造的目的，只好拉倒重來，重新洗牌，以洪水清洗一切生物與人類。

人犯罪為何旁及大地、生物？

(a) 始祖犯罪吃了禁果，結果地長出荊棘和疾藜，地必為你的緣故受咒詛，你必終身勞苦 ( 創3:17 ) ；

(b)該隱殺亞伯後，地開了口，現在你必從這地受咒詛，你種地，地不再給你效力，你必流離飄蕩在地上 ( 創4:11-12 ) ；

(c)  所多瑪和蛾摩拉城中人犯罪，耶和華降火，“把那些城和全平原，並城裏所有的居民，連地上生長的都毀滅了。( 創19:25 ) ；

(d) 人犯罪連累其他生物被毀

“我們知道一切受造之物，一同歎息，勞苦，直到如今”( 羅8:22 )

昔日如是，如今也是，現在的情況與洪水前差不多。

我們強暴、貪心、自私、偽善、欺壓、不法，不但對人，也對生物生境，

濫捕魚，破壞珊瑚、毀濕地，砍樹林、殺大象、斬犀牛、捕捉鯊魚、鯨魚

砍香樹、羅漢松…………….不一而足。

是否也會遭報呢？上帝昔日看不過眼，人先敗壞上帝才「毀滅」，

現在有自救之道嗎？

五、救贖之道

希望來自挪亞

「惟有挪亞在耶和華眼前蒙恩」( 創6:8 )

他有何特質可以蒙恩？他的特點是 ( 6:9 )

1.是個義人  不是惡人  (6:5)；不行強暴 ( 6:11) ；恩慈待人待物，合乎 法律和道德標準；

2.在當時的世代是個完全人

健全，遠離罪惡，獨特：

a.與後來的亞伯拉罕一樣 ( 創17:1 )

“你當在我面前作完全人”全心全意的奉獻和信靠；

b.約伯也有同樣特點：“完全正直，敬畏神，遠離惡事”( 伯1:1 )

c. 耶穌說“你們要完全，像你們的天父一樣”( 太5:48)

3.與神同行─如以諾

與神同行300年 ( 創5:22 ) ，明白神的思想和看法，因為挪亞的蒙恩，  蒙神指示建造方舟。與他立約，你同你的妻，與兒子，兒婦，都要入方舟( 6:18 )

4. 順服─凡神所吩咐的，他都照行了 ( 6:22 ) ，

當地上有血肉，有氣息的活物，無一不死之時 ( 6:17 ) ，神藉義人挪亞施  施施行拯救

施行拯救，因在這世代中，我見你在我面前是義人 ( 7:1 )

在當日世界敗壞時有挪亞，在今日世界同樣敗壞時，有誰？

挪亞尊守神吩咐可為生物留種 ( 7:3 ) ，但人類 － 特別是我們中國人，把留下來的都吃了，殺了，賣了，滅了，把生境都發展了！

六 結論  反思

環境保護的基礎

神身為創造者，對環境有關心，查看，關注，顧念，期望，有正義
     公義的存在；

人的罪行是環境破壞的主因；
不義的事影響大地和環境；
神因人的罪而毀滅大地；
也因一個義人而施行拯救；

也因愛護而與人、地、生物立約；
祂記念大地、人、生物，但人的表現令神失望，生物正步向滅種。
今天我們可以做甚麼?

看看世界真像，是否惡行，強暴、橫行；
了解人如何破壞世界生態；
在保護方面有不少可行的具體方法，主要的是效法挪亞，成為義人
在上帝眼中的完全人，與神同行─了解神的心意；順服─強過成功；

人與大地生物同是受迼者。

人被神委託為管理者，把大地、生物交付我們的手 (創9:2 ) ；因此我們有加倍的責任做好管家的職份。
